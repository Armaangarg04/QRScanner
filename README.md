# QR Security Scanner

A web application for scanning and generating QR codes with AI-powered security analysis.

## Features
- Scan QR codes using camera or upload images
- AI analysis of URLs for security threats
- Generate QR codes from text, URLs, or WiFi credentials
- Domain typosquatting detection
- Content safety analysis
